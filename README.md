## Implementation

Project under construction. Still in planning phases.

### Developing  
* `brunch watch` in `/frontend` (you will need to have [brunch](http://brunch.io) globally installed)  
* `npm start` in `/backend`  
* Application available at `localhost:8080`.  


### Dependencies
Bootstrap 3


Angular.JS


Node.JS
